Supported versions of Maya: 2011 and above

Using:
1. If Maya version is lower than  2014, first install PyQt for Maya
2. Copy folder pw_mGeo to the PYTHONPATH folder ( <mayaInstallFolder>\Python\Lib\site-packages\ )
3. In ScriptEditor execute code:

import pw_mGeo
pw_mGeo.show()
